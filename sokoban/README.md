A Simple Readme of This Easy Sokoban Game
====
大致思路
----
###
    其实就是每动一步就进行相应的判断
    然后对于前方有箱子的情况进行单独判断
    采取结构体的方式来存储特殊点的信息（比如起点 箱子 玩家还有目标地点）
    然后用一个函数打印屏幕
    一个函数控制移动
    一个函数判断玩家是否胜利
    
一些细节
----
###
    判断胜利采用的是判断箱子是否都到达目的地
    移动的判断包括前方有无箱子 前方有无障碍 箱子能不能动
    至于箱子是否到达目的地不需要判断
    因为实时更新了箱子的位置
